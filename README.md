# PinzaFlow

PinzaFlow is a WhatsApp CRM and AI-powered sales automation platform for small businesses in Latin America.

## Tech Stack
- **Frontend**: Nuxt 3 (Vue 3)
- **Styling**: Tailwind CSS
- **Backend**: Supabase
- **PWA**: @vite-pwa/nuxt
- **Icons**: Lucide Vue Next

## Getting Started

1. **Clone the project**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Set up Environment Variables**:
   Copy `.env.example` to `.env` and fill in your Supabase and OpenAI credentials.
4. **Initialize Database**:
   Run the SQL script in `supabase_schema.sql` in your Supabase SQL Editor.
5. **Run Development Server**:
   ```bash
   npm run dev
   ```

## Key Features
- **Conversations**: Manage WhatsApp chats in a modern interface.
- **Clients**: CRM to track customer data and tags.
- **AI Automation**: Placeholder for OpenAI-powered sales assistance.
- **PWA**: Installable on any device.
